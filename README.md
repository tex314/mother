## Clone repository

    $ git clone git@github.com:tex314/mother.git
    $ cd mother

## Install bundler

    $ gem install bundler

## Install rubygems

    $ bundle install

## Launch the server

    $ middleman server
    == The Middleman is loading
    == The Middleman is standing watch at http://0.0.0.0:4567

## Check the site

    $ open http://0.0.0.0:4567 

## Add a page

    $ emacs source/20130401.html.markdown
